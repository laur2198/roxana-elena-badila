# Imagini

Locul pentru imaginile site-ului. De adăugat de către proprietar/dev:

- `og-default.svg` — imagine OG implicită (placeholder; ideal înlocuiește cu un `.jpg`/`.png` 1200×630 real și actualizează `SITE.defaultOgImage` în `src/consts.ts`).
- portret/fotografii cabinet — pentru hero și /despre (înlocuiesc casetele placeholder din pagini).
- `blog/` — imagini de cover pentru articole (referite în frontmatter prin `cover:`).

Recomandare: format WebP/AVIF, dimensiuni optimizate, cu `alt` descriptiv.
